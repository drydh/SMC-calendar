Needs Python 2 installed with bs4 (BeautifulSoup)

Usage: 
1. Make sure the calendar, semads.py and seminarmailer.py files are all in the same directory.
2. Create a directory called Archive (if it doesn't already exist) within the above directory. 
3. Update "dir" in the "calendar" file to reflect the directory containing these files. 
4. Update "user" in the "calendar" file to reflect your KTH username.
5. Make sure that the calendar file is executable (e.g. use chmod).
5. Run ./calendar from the directory containing the files. 
6. The script will create the week's email and display it in the nano editor. 
7. Check the email and press ctrl+o to save any changes.
8. When the email has been checked press ctrl+x.
9. Press y to send the email or ctrl+c to exit. 
10. When promtped, type in your KTH password to send the email. 

Notes:
- You can run these files from any computer. 
- If you are running these files from your own computer there is an option to use socks proxies for the mailer.
